# 自媒体管理应用

一个用于管理自媒体推广记录的应用，支持记录账号名、推荐产品、推广日期和收益信息。

## 功能特性

- ✅ 表单输入：可以添加新的推广记录
- ✅ 数据持久化：使用 LocalStorage 保存数据
- ✅ 日程日历：按日期查看和管理推广记录
- ✅ 收益看板：统计和分析收益数据
- ✅ 响应式设计：支持桌面和移动设备

## 数据结构

每条记录包含以下字段：
- **账号名**：自媒体账号名称
- **推荐产品**：推广的产品名称
- **推广日期**：推广日期
- **收益**：收益金额（数字）

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

4. 预览生产版本：
```bash
npm run preview
```

## 使用说明

1. **添加记录**：在表单中填写账号名、推荐产品、推广日期和收益，点击"保存记录"按钮。

2. **日程日历**：
   - 选择日期查看该日期的所有记录
   - 查看所有记录的列表
   - 可以删除不需要的记录

3. **收益看板**：
   - 查看总收益、记录总数和平均收益
   - 按账号统计收益情况
   - 按产品统计收益情况
   - 可以设置日期范围进行筛选

## 技术栈

- React 18
- Vite
- LocalStorage API
- CSS3 (响应式设计)
